Determines if the provided attribute has changed its value.

The logic in the function will first determine the attribute to check using the attribute parameter and then will call the appropriate function to check if the attribute did change.
Parameters
file_idThe id of the file.
attributeThe attribute to check.
tableThe table to check based on whether the attribute is in the changes or the files table.
dbThe sqlite3 database object.
char_attrThe character value attribute to check. The default is NULL.
int_attrThe integer value attribute to check. The default is 0
